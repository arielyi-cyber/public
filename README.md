# Employee Financial Tracker

A beginner C++ application that allows users to enter employee hours worked and pay rate information to calculate earnings.

## Features
- Accepts employee input
- Calculates wages based on hours worked
- Displays payroll information
- Demonstrates basic C++ programming concepts

## Technologies
- C++
- GitHub

## Purpose
Created as a programming assignment to practice user input, calculations, variables, and output formatting in C++.
